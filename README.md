## 📦 What this is

Skills is a GitHub repository that stores reusable skills for AI coding agents. It uses a multi-skill workspace, so more skills can be added over time without making a new repository for each one.

If you want a simple way to keep AI agent tasks organized, this repo gives you a clean place to do that. It also helps you manage skill files in one spot and keep them easy to find.

## ✨ What you get

## 🔧 What `cross-agent-skill-sync` does

This skill helps with skill management across agent tools. It can help you:

- compare skill sets across tools
- find skills that are missing in one place
- check which skills are only covered outside the main set
- plan symlink changes before making them
- sync existing skills into the right place
- remove target-side links safely

This is useful when you want your skill folders to stay in sync across more than one agent setup.
